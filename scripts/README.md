## 🔧 Personalização

### **Alterar Configurações**

Edite as variáveis no início do script:

```bash
# Configurações do sistema
BACKEND_PORT=8081
FRONTEND_PORT=3000
MAX_PROCESSES=50
SUPER_LOAD_INTERVAL=5
```

### **Ajustar Carga**

- **Aumentar carga**: Aumente `MAX_PROCESSES`
- **Reduzir carga**: Diminua `MAX_PROCESSES`
- **Mais rápido**: Diminua `SUPER_LOAD_INTERVAL`
- **Mais lento**: Aumente `SUPER_LOAD_INTERVAL`
