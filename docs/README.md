## Tecnologias Documentadas

### Backend

- **Java 17** - Linguagem principal
- **Spring Boot 3.2.0** - Framework de aplicação
- **Spring Data JPA** - Persistência de dados
- **H2 Database** - Banco de dados em memória
- **Maven** - Gerenciamento de dependências

### Frontend

- **React 18** - Biblioteca de interface
- **TypeScript** - Tipagem estática
- **Material-UI** - Componentes de interface
- **Google Maps API** - Mapas interativos

### Algoritmos

- **Genetic Algorithm** - Otimização evolutiva
- **Simulated Annealing** - Otimização por recozimento
- **K-means Clustering** - Agrupamento de dados
- **TSP (Traveling Salesman Problem)** - Problema do caixeiro viajante

### Deploy e DevOps

- **Cloudflare Pages** - Deploy de frontend
- **Cloudflare Workers** - Deploy de backend
- **GitHub Actions** - CI/CD
- **Docker** - Containerização
